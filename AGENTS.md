# Java Playground

## WHY
A personal sandbox for exploring Java language features, experimenting with new syntax, trying out libraries, and running quick experiments.

## WHAT
- Java 25 project using Gradle
- Collection of experimental code and learning exercises
- Test-driven exploration of language features and libraries

## HOW
- **Build**: Gradle
- **Java Version**: 25
- **Testing**: Comprehensive tests are important - validate experiments with good test coverage
- **Design**: Follow object-oriented design principles
- **Style**: Experiment freely, but keep code clean and well-structured
